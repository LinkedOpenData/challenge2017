検索例（SPARQLスニペット）

※上述の「SPAQRLエンドポイント」のクエリ入力欄に下記のクエリをコピーして実行すると結果をご覧いただけます．

(1) 任意のトリプルを100件取得 
SELECT * WHERE {
  ?s ?p ?o 
}LIMIT 100


(2) 指定した作品 　（この例ではグラフURI＝http://lod.sfc.keio.ac.jp/challenge2011/show_status.php?id=d003） 　から任意のトリプルを100件取得 
SELECT * WHERE {
 GRAPH <http://lod.sfc.keio.ac.jp/challenge2011/show_status.php?id=d003> {
    ?s ?p ?o 
 }
} LIMIT 100
※「Virtuoso SPARQL Query Editor」のDefault Data Set Name (Graph IRI)欄にグラフURIを入力すれば，(1)と同じクエリで実行可能

(3) 応募作品のグラフURIとタイトル一覧を100件取得 
SELECT ?id ?title WHERE {
 GRAPH <http://lod.sfc.keio.ac.jp/challenge/entries> {
    ?s <http://purl.org/dc/terms/title> ?title;
       <http://www.w3.org/2000/01/rdf-schema#seeAlso> ?id. 
 }
} LIMIT 100
